tomdoc.sh
=========

tomdoc.sh will parse [TomDoc] embedded in shell scripts and generate pretty
documentation from it. Eventually.

[TomDoc]: http://tomdoc.org


Usage
-----

    $ tomdoc.sh some_script.sh
